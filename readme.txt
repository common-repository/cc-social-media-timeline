== Description ==

CC Social Media Timeline is a plugin to show your social media feeds on your website.

Major Features:<br>
1. Simple Shortcode.<br>
2. Show facebook timeline.<br>
2. Show twitter timeline.<br>

How to use: <br>

For Showing Favebook Timeline: <br>

Shortcode: <pre>[cc_fb_timeline]</pre> <br>
Options:
<ul>
	<li><b>page_id: </b>Insert your facebook page id with this attribute.</li>
	<li><b>page_title: </b>Use to show the timeline title name.</li>
	<li><b>small_header: </b><i>(default: false)</i> Use to make the timeline page header image bigger/smaller. <i>(values: true/false)</i></li>
	<li><b>cover: </b><i>(default: false)</i> Use to show the timeline cover image. <i>(values: true/false)</i></li>
</ul>
Full Shortcode : <pre>[cc_fb_timeline page_id="yourpageid" page_title="Page Title" small_header="false" cover="false"]</pre>
<br>

For Showing Twitter Timeline: <br>

Shortcode: <pre>[cc_twt_timeline]</pre> <br>
Options:
<ul>
	<li><b>page_id: </b>Insert your twitter page id with this attribute.</li>
</ul>
Full Shortcode : <pre>[cc_twt_timeline page_id="yourpageid"]</pre>
<br>

Plugin Demo: http://www.codycave.com/Plugins/cc-social-media-timeline/


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `CC Social Media Timeline` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can use shortcode in page or post.
4. After applying the shortcode please go to the endless scroll plugin option (in your wordpress Dashboard Menu-bar) and click on 'Save Changes' button once.
